# 微信云托管智能写稿下单系统

支持：
- 质感前端页面（玻璃拟态风格）
- AI助手在页面最上部，不影响下单区域
- 客户输入昵称并存储到后台
- 支持上传 `.docx` 或 `.txt`，也支持纯文本直接输入
- 字数上限 30000
- 自动计价函数（按你的规则）
- 清晰后台管理页，便于提取订单信息
- 适配微信云托管（Docker + `PORT` 环境变量）

## 计价规则
- 写稿费：250字 = 1元
- 纸费：
  - 非 A4：3000字 = 1元
  - A4：1000字 = 1元
- 字体：可选 `楷宋体` / `行草体` / `宋楷体`
  - `宋楷体` 每1000字额外 +2元
- 加急：一日内成品，总价 × 2.5

## 本地运行
```bash
npm install
npm start
```
打开：
- 前台下单页：`http://localhost:3000/`
- 后台管理页：`http://localhost:3000/admin.html`

## AI模型接入（免费可选）
默认兼容 OpenAI 风格接口（已内置 OpenRouter 示例）：
- `OPENAI_API_KEY`
- `OPENAI_BASE_URL`
- `OPENAI_MODEL`

未配置 Key 时，会走“演示模式”，可正常流程调试。

## 微信云托管部署建议
1. 将项目推送到你的 Git 仓库。
2. 在微信云托管中新建服务并关联仓库。
3. 使用根目录 `Dockerfile` 构建镜像。
4. 设置环境变量（至少可设置 `PORT=80`，并按需设置 AI 相关变量）。
5. 发布后即可获得可访问网址。

## 数据说明
- 使用本地 `data.json` 存储订单（字段含昵称、文本、AI内容、金额、时间等）。
- 后台页 `/admin.html` 可直接查看订单列表。

> 生产建议：可把 `data.json` 改为云数据库（MySQL/PostgreSQL），避免单实例文件存储限制。
